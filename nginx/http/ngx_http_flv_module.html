Module ngx_http_flv_module

Example Configuration
Directives
     flv

ngx_http_flv_module模块为Flash Video（FLV）文件提供伪流服务器端支持。

它特别通过请求URI的查询字符串中的start参数处理请求，方法是从请求的字节偏移量和前置FLV头开始发回文件的内容。

默认情况下不构建此模块，应使用--with-http_flv_module配置参数启用它。

示例配置 

location ~ \.flv$ {
    flv;
}

指令

Syntax: flv ;
Default: —
Context: location

打开周围位置的模块处理。
