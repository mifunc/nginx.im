Module ngx_http_index_module

Example Configuration
Directives
     index

ngx_http_index_module模块处理以斜杠字符（' / '）结尾的请求。 这些请求也可以由ngx_http_autoindex_module和ngx_http_random_index_module模块处理。

示例配置 

location / {
    index index.$geo.html index.html;
}

指令

Syntax: index file ...;
Default: 
index index.html;
Context: http , server , location

定义将用作索引的文件。 file名可以包含变量。 按指定顺序检查文件。 列表的最后一个元素可以是具有绝对路径的文件。 例： 

index index.$geo.html index.0.html /index.html;

应该注意，使用索引文件会导致内部重定向，并且可以在不同的位置处理请求。 例如，使用以下配置： 

location = / {
    index index.html;
}

location / {
    ...
}

“ / ”请求实际上将在第二个位置处理为“ /index.html ”。
