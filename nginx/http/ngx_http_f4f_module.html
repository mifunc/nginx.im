Module ngx_http_f4f_module

Example Configuration
Directives
     f4f
     f4f_buffer_size

ngx_http_f4f_module模块为Adobe HTTP动态流（HDS）提供服务器端支持。

该模块以“ /videoSeg1-Frag1 ”形式实现HTTP动态流式处理请求的处理 - 使用videoSeg1.f4x索引文件从videoSeg1.f4f文件中提取所需的片段。 该模块是Apache的Adobe f4f模块（HTTP Origin Module）的替代品。

需要使用Adobe的f4fpackager进行常规预处理，有关详细信息，请参阅相关文档。

该模块作为我们商业订阅的一部分提供。

示例配置 

location /video/ {
    f4f;
    ...
}

指令

Syntax: f4f ;
Default: —
Context: location

打开周围位置的模块处理。

Syntax: f4f_buffer_size size ;
Default: 
f4f_buffer_size 512k;
Context: http , server , location

设置用于读取.f4x索引文件的缓冲区的size 。
