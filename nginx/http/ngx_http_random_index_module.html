Module ngx_http_random_index_module

Example Configuration
Directives
     random_index

ngx_http_random_index_module模块处理以斜杠字符（' / '）结尾的请求，并在目录中选择一个随机文件作为索引文件。 在ngx_http_index_module模块之前处理该模块。

默认情况下不构建此模块，应使用--with-http_random_index_module配置参数启用它。

示例配置 

location / {
    random_index on;
}

指令

Syntax: random_index on | off ;
Default: 
random_index off;
Context: location

启用或禁用周围位置的模块处理。
