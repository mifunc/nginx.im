Building from the sources

作为Linux软件包安装

对于Linux，可以使用njs模块包 ：

  • nginx-module-njs - njs 动态模块
  • nginx-module-njs-dbg - nginx-module-njs包的调试符号

从源头构建

可以使用以下命令克隆具有njs源的存储库 :(需要Mercurial客户端）： 

hg clone http://hg.nginx.org/njs

然后应使用--add-module配置参数编译--add-module ： 

./configure --add-module= path-to-njs /nginx

模块也可以构建为动态 ： 

./configure --add-dynamic-module= path-to-njs /nginx

要仅构建njs命令行实用程序 ，请运行./configure并从njs根目录make njs命令。 该实用程序以./build/njs 。
